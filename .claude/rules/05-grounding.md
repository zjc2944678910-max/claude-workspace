# Grounding 补充规则

完整的验证优先、事实分类、环境边界规则见 workspace CLAUDE.md。以下仅补充 CLAUDE.md 未覆盖的规则。

## 环境可达性速查
- 执行命令前，确认目标路径和工具在当前环境中可用。
- macOS 临时目录、GUI app 状态、Finder 位置——视为宿主机侧，不在 Cowork shell 范围内。
- 如果命令必须在真实 Mac 终端执行，明确告知用户。

## 中文不确定性短语
遇到不确定时优先使用：
- `我还没有验证这个路径`
- `我还没有验证这个状态`
- `这个内部行为目前不可见`
- `这看起来是宿主机路径，不一定能从 Cowork shell 访问`
- `这个操作更适合在真实 Mac 终端里执行`
- `当前证据不足以确认根因`
- `根据当前证据只能确认到这里`
- `下一步可以通过 ... 验证`
