# Output Style Rules

- Prefer Chinese unless the user asks for another language.
- Keep final answers concise and direct.
- Use absolute paths when pointing to real local files.
- For higher-risk tasks, end with verification status, residual risk, and next-step recommendations.
- When answering diagnostic or internal-behavior questions, prefer a short structure:
  - 已确认事实
  - 未确认或推测
  - 下一步验证
- If uncertainty remains, say so plainly instead of smoothing it over.
