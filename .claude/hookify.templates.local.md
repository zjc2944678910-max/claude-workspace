# Template Awareness

When creating these document types, use the matching template:

| Document type | Template path |
|---------------|---------------|
| Audit report (L2 诊断报告) | `context/templates/audit-report.md` |
| Repair plan (L3 修复方案) | `context/templates/repair-plan.md` |
| Cleanup manifest (清理计划) | `context/templates/cleanup-manifest.md` |
| Project note (项目笔记) | `context/templates/project-note.md` |
| Long task run (多步骤任务) | `context/templates/long-task-run.md` |

Read the template first. Follow its structure. Fill sections relevant to current task.
